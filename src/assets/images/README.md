# Logo Assets

Please place the Stryvon logo file here with the following name:

- **stryvon-logo.svg** - The SVG logo used for selfAccounting.AI application

## File Location

Place your SVG logo file at:

```
frontend/smart-expense-uae/src/assets/images/stryvon-logo.svg
```

## Logo Usage

The logo is used in two places:
1. **Login Page** - Displayed in the auth card avatar (circular, ~48px with padding)
2. **Sidebar** - Displayed in the brand section (40x40px with 6px padding)

The SVG logo will automatically scale and maintain its aspect ratio.

## File Format

- Format: SVG (Scalable Vector Graphics)
- Recommended: Optimized SVG file
- Should work with transparent background
- Will be displayed with `object-fit: contain` to preserve aspect ratio
