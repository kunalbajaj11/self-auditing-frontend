User Roles & Permissions

Admin, Accountant, Approver, Auditor, Employee: what each role can see and do

Roles Overview

Admin

Full access within the organization: manage users, expenses, categories, expense types, ledger accounts, all reports, bank reconciliation, organization settings, sales invoices, customers, vendors, purchase orders, payments, journal entries.

Accountant

View and manage all expenses, create/edit expenses, manage categories, run financial reports, bank reconciliation, accruals. Cannot manage users or organization settings.

Approver

View expenses pending approval, approve or reject expenses, view reports. Cannot create expenses or manage categories/settings.

Auditor

Read-only: view all expenses and reports, access audit logs, generate audit reports. Cannot create or edit data.

Employee

Create and manage own expenses, upload receipts, view own expenses only. Cannot view others’ expenses or access reports.

Plan Limits

Standard: Up to 5 users. Premium / Enterprise: Unlimited users. When limit is reached, “Invite User” is disabled until upgrade.
