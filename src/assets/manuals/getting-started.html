Getting Started & Quickstart

First-time setup: register, configure, and add your first expense • 30–60 min

Overview

Welcome to SelfAccounting.AI. You will need: license key, company details (name, VAT, address), admin name/email/password, and bank details.

1. Register with License Key

  1. Homepage → Get Started or Register with License.
  2. Enter license key → Validate.
  3. Fill form: Organization Name, VAT, Address, Currency (e.g. AED), Region, Admin Name, Email, Password (min 8 chars).
  4. Click Register. You are logged in; welcome email sent. Organization and default categories are created.

2. Essential Configuration

Settings → Organization Settings — Company name, TRN, currency, contact, address, bank details.

Settings → Tax Settings — Add VAT rate (e.g. 5% UAE), zero-rated/exempt if needed.

Settings → Invoice Template — Logo, footer, branding.

Settings → Numbering Sequences — Invoice prefix (e.g. INV-2025-), expense reference.

3. Master Data (Before First Expense)

Settings → Expense Categories — Review/add categories.

Settings → Expense Types — OPEX, CAPEX, link to ledger.

Settings → Ledger Accounts — Chart of accounts.

Banking → Bank / Cash Accounts — Add bank and petty cash.

Contacts → Vendors — Add vendors (expenses link to vendor).

Contacts → Customers — Add customers if you issue invoices.

4. User Management

Settings → User Management — Invite team; assign Admin, Accountant, Employee. Standard plan = 5 users max.
