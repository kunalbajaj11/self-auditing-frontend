Migration: Tally & Zoho Books → SelfAccounting.AI

Move opening balances using trial balance import

Steps

  1. Cutover date — e.g. 1 Jan 2025 or start of fiscal year.
  2. Export — From Tally: Display → Trial Balance → Export. From Zoho: Reports → Trial Balance → Export. Date = day before cutover. Columns: Account Name, Debit, Credit.
  3. Setup in SelfAccounting.AI — Complete registration and setup; create custom ledger accounts if needed; ensure Bulk Import is enabled.
  4. Import — Expenses → Journal Entries → Bulk Import. Upload CSV, preview mapping, confirm total debits = total credits, Import.
  5. Verify — Run Trial Balance in SelfAccounting.AI; compare to exported trial balance.
  6. Go live — Record all new transactions in SelfAccounting.AI from cutover date.

Account Mapping

Use owner_shareholder_account (not Retained Earnings) for opening balance equity. Map Tally/Zoho account names to SelfAccounting.AI codes (cash, bank, accounts_receivable, accounts_payable, vat_receivable, vat_payable, etc.). See Data Import Templates for the full table.
