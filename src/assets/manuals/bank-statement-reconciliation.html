Bank Statement Import & Reconciliation

Upload bank statements and match transactions

Overview

Upload a statement (e.g. CSV), then match each statement line to the correct transaction (payment, expense, journal entry) or mark as matched/unmatched.

Step 1: Add Bank Accounts

Banking → Bank Accounts — Add each account (name, currency, optional opening balance).

Step 2: Upload Statement

Banking → Upload Bank Statement — Select account, choose CSV (or supported format), upload. System parses and shows preview. Confirm column mapping if offered, then Import.

Step 3: Reconcile

Banking → Reconciliation — Select account and statement. Match each statement line to a transaction or mark as matched. When debits/credits match statement total, reconciliation is complete.
