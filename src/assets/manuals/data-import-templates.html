CSV formats for opening balances, ledger accounts, and journal entries

Opening Balances (Trial Balance)

CSV columns: Account Name (or Code), Debit, Credit. Use owner_shareholder_account as balancing account. Expenses → Journal Entries → Bulk Import (when enabled).

Ledger Accounts

If bulk import is available: CSV with Name, Category (asset/liability/equity/revenue/expense). Create custom accounts before journal import if you reference them by ledger:{uuid}.

Where to Import

Opening balances / journal entries: Expenses → Journal Entries → Bulk Import. Ledger accounts: Settings → Ledger Accounts → Import (if available). Contact your account manager if Bulk Import is not visible.
